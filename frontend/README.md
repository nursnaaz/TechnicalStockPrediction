# Bullish Stock Scanner - Frontend

React + TypeScript frontend for the Bullish Stock Scanner MVP, built with Vite and Cloudscape Design System.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs at http://localhost:5173/

## Build

```bash
npm run build
```

Outputs to `dist/`

## Preview Production Build

```bash
npm run preview
```

## Environment Variables

Create a `.env` file (optional):

```bash
VITE_API_URL=http://localhost:8000
```

Default backend URL is http://localhost:8000

## Tech Stack

- **React 19** - UI framework
- **TypeScript 6** - Type safety
- **Vite 8** - Build tool and dev server
- **Cloudscape Design System** - AWS-style UI components
- **ESLint** - Code linting

## Project Structure

- `src/App.tsx` - Main application component
- `src/main.tsx` - React entry point
- `src/components/` - React components
- `src/services/` - API client services
- `src/types/` - TypeScript type definitions
- `src/styles/` - CSS styles

Project initialized and ready for component development. Basic structure in place with Cloudscape Design System configured.
